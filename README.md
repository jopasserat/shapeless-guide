## Reading the Book

An up-to-date PDF should can be found in [dist/shapeless-guide.pdf][pdf].

## Building the Book

Install Docker and use `go.sh` to boot an instance
with all the right dependencies:

~~~
bash$ ./go.sh
~~~

Then use `sbt` to build the book:

~~~
npm install grunt
npm install
sbt pdf
~~~
